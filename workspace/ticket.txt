TICKET
======
Summary: The script contains logic errors in list manipulation functions, causing incorrect flattening depth, incomplete running totals, and missing list chunks.

Bugs found:
1. Function: flatten
   Problem: The `depth` parameter is ignored, causing the function to always completely flatten the list, regardless of the specified depth.
   Expected: The function should flatten the `nested_list` only up to the specified `depth`. For `depth=1`, it should only flatten the first level of nesting.

2. Function: running_total
   Problem: The loop iterates from `transactions[2:]`, skipping the second element (`transactions[1]`) from the total calculation.
   Expected: The function should include all elements from `transactions[1]` onwards in the running total.

3. Function: chunk_list
   Problem: The loop condition `len(lst) - size` causes the function to prematurely stop iterating, resulting in the last chunk(s) of the list being omitted from the `chunks` list.
   Expected: The function should include all elements of the original list, creating chunks of the specified `size`, with any remaining elements forming a final, potentially smaller, chunk.

End of ticket.
